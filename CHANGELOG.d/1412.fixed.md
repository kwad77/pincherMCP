`pincher bench --project` now accepts NAME or substring (not just the literal stable ID), matching the tiered match in `pincher project rm`, `pincher verify --project`, and `pincher doctor --project`. Pre-fix bench was the only project-arg subcommand requiring an exact ID — passing `--project pincher-repo` would fail with "no symbols with edges in project pincher-repo" even when pincher-repo had 12k+ edges, because the literal "pincher-repo" never matched the stable id `d:\claudecode\pincher-repo`. Ambiguous matches refuse to silently pick one and list the candidates. (#1412)
