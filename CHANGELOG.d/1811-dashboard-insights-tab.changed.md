The HTTP dashboard's Overview tab is split into Overview + a new Insights tab ([#1811](https://github.com/kwad77/pincher/pull/1811)). The Overview had grown to 11 stacked single-purpose panels with no hierarchy — the headline token-savings ROI scrolled past alongside niche diagnostics. The 6 deep diagnostics (backend status, tool-call breakdown, complexity-tier breakdown, response-payload-size, tool-mix entropy, bench history) now live on an Insights tab; Overview keeps the ROI story (session / all-time savings / sparkline) and the PreToolUse hook panel. The redundant `last-refresh` header badge is dropped (the `updated-ago` badge already shows refresh recency). No new endpoints — same data, clearer information architecture.
