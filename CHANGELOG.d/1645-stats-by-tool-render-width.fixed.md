`pincher stats` BY TOOL section now renders with column-aligned rows that fit inside the box border (#1645 v0.86). Pre-fix the v0.85 row format (`"1×, 4500ms tot, 4500ms avg, 4500ms max"`) was 38+ chars and overflowed the 44-char inner width — the right border abutted the text without padding. New format: header row + four numeric columns (count, total, avg, max) padded to fixed widths; values ≥10 seconds collapse to `Ns`, values ≥60 seconds collapse to `Nm` so columns stay narrow without information loss. Tool names >12 chars truncate with `…` (defensive — no current tool name exceeds 12). Width-snapshot test pins the invariant that every box line is the same length.
